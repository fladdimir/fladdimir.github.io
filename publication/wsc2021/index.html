From Logistics Process Models to Automated Integration Testing: Proof-of-Concept using Open-Source Simulation Software

Wladimir Hofmann, Paul Reichardt, Sebastian Lang, Tobias Reggelin
December 2021
Code Slides Video
wsc

Abstract

This paper explores the practical integration of simulation methods into software development processes. It shows an automated integration testing approach, which enables continuous virtual commissioning. Following an analysis of the current state of knowledge and the standards of software development, a case study on logistics order management is presented, referring to a typical B2B application in the retail logistics sector. The proof-of-concept shows how the usage of a simulation model for automated integration testing and its inclusion into continuous-integration can help to ensure software quality, particularly for process-centered logistics applications. The implemented setup proves the feasibility of the approach, using standard open-source development tools and a Python-based open-source simulation library.
